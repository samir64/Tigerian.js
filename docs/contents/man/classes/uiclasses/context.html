Description:


The `Context` constructor gives four parameters:
First and last parameters (`parent` and `theme`), are like other controls.
Third parameter (`route`) is an instance of Tigerian's Route class, and it is optional. If you set that, your application can change address without refresh and if you don't set that, it's a normal link which after clicking on link, page get a refresh.
Most important parameter in `Context` is second parameter (`text`).
In `text` we can use three exclusive tags:

  • Variable:


    <variable id="v1">default value</variable>

  • Control:


    <control id="c1">head text|foot text</control>

  • Redirect:


    <redirect url="/test">redirect to test page</redirect>

You can access to variables and controls in your `text`, by `entities` property:

Sample:


ctx1.entities.v1 = "hello";
ctx1.entities.c1.addControl(new Control());


Examples:


let app = new Application("title");
let route = new Route("/");
let ctx1 = new Context(app, "sample text for <variable id="varCtrlName">context</variable> control", route);

ctx1.entities.varCtrlName = "Tigerian's Context";
